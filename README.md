# 🔥 RoastMyCode

> Paste your code. Get destroyed. Actually fix it.

An AI-powered code roaster that judges your code like a savage stand-up comedian — then gives you real fixes.

## Features
- 🎤 Brutal comedian-style roast of your code
- 🐛 Issue detection with severity levels
- 🔧 Concrete fixes and improved code snippets
- 💀 Hall of Shame leaderboard
- 🎴 Shareable roast cards
- 10 languages supported

## Stack
- Vanilla React (via CDN) — zero build step
- Anthropic Claude API
- Single `index.html` — deploys anywhere instantly

## Deploy
Just push to GitHub and connect to Vercel. No build config needed.

Built at v0 India IRL Hackathon 🇮🇳
